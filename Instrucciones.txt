Hola Raul!!! este es un tutorial para que puedas instalar el programa y se lo puedas mostrar al profesor Enriquez

1.- crea una carpeta en el ESCRITORIO de tu computadora y llamala Reconocimiento
2.- Descarga el codigo desde este link https://github.com/JairoAlan/Recono_JR le das clic en code (boton verde) y en Download Zip
3.- descomprimes el programa dentro de la carpeta que creaste en el ESCRITORIO
4.- Abre el Visual studio Code, desppues le das click en File o archivo, y abrir un Folder o Open folder, y abres el folder que creaste en tu ESCRITORIO
5.- Abres una terminal en Terminal> Nuevo Terminal Creas el ambiente virtual: virtuaenv venv "O el nombre que quieras"
6.- Activas el ambiente desde la terminal con el comando ./"venv" 'o el nombre que eligiste'/Scripts/activate
7.- Sabras que ya se activo por que en la terminal te aparece el nombre del ambiente
8.- Asegurate que el contenido del programa este dentro de la carpeta ESCRITORIO
9.- Haz click en la Terminal y ejecutas lo siguiente: pip install -r requirements.txt
10.- Despues de que se instale todo lo necesario, puede ejecutar el comando en la terminal: uvicorn app:app --reload
11.- En la terminal se mostrara distintos valores:
Will watch for changes in these directories: ['C:\\Users\\Su Usuario\\Desktop\\ambiente']
INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit) <----------Copian esta direccion url y la pegan en el navegador
INFO:     Started reloader process [27852] using WatchFiles
INFO:     Started server process [71228]
INFO:     Waiting for application startup.
INFO:     Application startup complete.

12.- y en teoria ya quedo les mostra la aplicacion en su browser.
